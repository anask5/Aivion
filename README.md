# 🌌 AI Study Assistant - Aivion

> Your intelligent study partner powered by AI 🚀  
> Chat • Summarize • Focus • Learn smarter

---

## ✨ Features

- 💬 **AI Chatbot** – Ask questions and get instant answers  
- 📝 **Summarizer** – Convert long notes into short insights  
- ⏳ **Study Timer (Pomodoro)** – Focus with galaxy UI + lofi mode 🎧  
- 🧠 **Quiz Generator** – Practice with AI-generated questions  
- 📇 **Flashcards** – Quick revision tool  
- 📱 **Responsive UI** – Works on all devices

## 🌠 UI Highlights

- 🌌 Galaxy-themed interface  
- ✨ Glowing neon elements
